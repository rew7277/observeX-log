# V34 Noise Reduction / Enterprise Clarity Updates

This build focuses on reducing dashboard noise and improving decision clarity.

## Updated
- Masking Settings now groups rules into Personal Data, Financial Data, Identifiers & Keys, and System Fields.
- Masking Settings supports enabled-count summary and collapsible groups.
- Findings now show summary cards and key insights first, with raw evidence collapsed.
- Smart Tags are grouped into Flows, APIs, Errors, Infra, IDs, and Other with tag limits.
- API Registry now collapses duplicated processor/event rows into grouped API entries.
- API Registry shows raw detected rows only inside expandable details.
- Long names/messages are clipped for readability.
- Top Error Clusters and next steps are limited to actionable items.

## UX Rule Applied
Summary first → insight second → detail on demand.
